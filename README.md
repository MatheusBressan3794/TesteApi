Este script usa a API da WeatherAPI. Para funcionar, você precisa criar uma conta gratuita e gerar sua API Key.

Siga exatamente estes passos:

Acesse o site oficial:
https://www.weatherapi.com/

Clique em Sign Up (Criar conta) e faça seu cadastro gratuito.
Após criar a conta e fazer login, vá ao seu painel (Dashboard).
Copie a API Key que aparece na sua conta.
Cole essa chave no código, na linha:
api_key = "SUA_CHAVE_AQU"

Esse programa vai devolver as informações de temperatura e descrição do tempo
Se quiser ver outras funcionalidades da API, acesse a documentação oficial:
https://www.weatherapi.com/docs/
